# PySoundLab

A Python project for sound processing and analysis.

## Setup

1. Ensure Python 3.8+ is installed.
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

Run the main script:
```
python main.py
```

## Features

- Sound recording and playback
- Audio analysis
- Visualization of waveforms
